Une petite introduction ...

Mirande 2.0, est un site web réalisé pour le projet de fin de semestre du cours d'Info 1B: programmation web, de monsieur de Garnier.
Ce site comporte cinq page web, qui a pour but dans un premier temps de montrer les compétences acquises au cours de ce semestre dans les différents langages tel que l'HTML, le CSS et le JavaScript.
Et dans un deuxième temps de ce site permet de vous faire découvrir Mirande et l'université du point de vu des étudiants dans la page "Les conseils d'Anon", les différentes expériences d'une étudiante dans "Le coin de Merclotte".
La page "Que savons nous faire ?" Traite de sujet plus technique avec une calculatrice, et un dessin réaliser avec des courbes de Bézier.
Pour finir "Contacter-nous" est un formulaire que vous pouvez remplir. Le bouton 'Valider' permet de vérifier que le formulaire est bien rempli, que l'adresse e-mail 1 est valide du point de vue de la syntaxe, et que les deux adresses e-mail sont identiques. Le bouton 'Envoyer' permet d'envoyer ce formulaire sur une adresse e-mail personnel grâce à une fonction PHP.
